# AMXX EBOT No Spread No Recoil (ReAPI)
This AMX Mod X plugin was made for E-BOT 1.07 LEGACY R12 but works for any bots.

It makes bots to have laser accuracy when running and shooting zombies.

[E-BOT Legacy R12](https://github.com/EfeDursun125/CS-EBOT-LEGACY)

# CVars
- ebot_aim_nospread 1 // Bots bullets do not spread, they go dead center in their crosshair
- ebot_aim_norecoil 0 // Bots crosshair does not move/kick when shooting

# Requirements
ReAPI (https://github.com/rehlds/ReAPI)

# Note
E-BOT Legacy R12 is known to be the most stable version to use with Zombie Mods in CS 1.6.
